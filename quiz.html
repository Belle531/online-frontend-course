What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages...


                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <title>My First Webpage</title>
                </head>
                <body>
                    <h1>Hello World!</h1>
                    <p>This is my first paragraph.</p>
                </body>
                </html>
